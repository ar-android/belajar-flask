Kenapa Belajar Flask

Saat ini banyak sekali bahasa pemrogramman yang bisa kita gunakan untuk membuat sebuah aplikasi web, di indonesia sendiri yang paling populer adalah PHP, lalu kenapa kita harus belajar Flask?. Berikut ini mungkin sedikit alasan yang bisa anda pertimbangkan untuk memulai belajar flask.

  • Flask dibangun diatas bahasa pemrogramman python.
  • Python adalah bahasa pemrogramman yang sangat mudah dipelajari.
  • Sintak python sangat mudah dimengerti bagi programmer pemula.
  • Secara peforma python cukup bersaing dengan bahasa pemrogramman moder yang sekarang banyak bermunculan, bahkan sekelas instagram saja mereka masih menggunakan python sebagai bahasa backendnya.
  • Bagi anda yang masih pemula dengan python maka flask adalah pilihan terbaik untuk memulai belajar membuat web dengan python.
  • Dan jika kamu ingin bekerja remote, maka dengan menguasai bahasa pemrogramman python maka sangat mudah bagikamu untuk bekerja dengan bayaran Rp. 100.000 / jam.
