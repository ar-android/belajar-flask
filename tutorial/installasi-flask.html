Installasi Flask

Apa saja yang perlu di install untuk memulai mempelajari Flask.

Flask

Pada tutorial sebelumnya kita sudah bahas panjang lebar cara install python di beberapa sistem operasi. Selanjutnya kita perlu install flask, nah flask ini adalah sebenarnya kumpulan kode, jadi pada proses installasi flask kita akan mendownload code flask ini ke komputer kita.

Installasi Python

Proses installasi python pada sistem operasi windows, linux dan osx.

Pada python untuk pengolahan dependency code kita kenal dengan nama pip. Ok sekarang yang pertama kita install virtual env terlebih dahulu dengan menjalankan perintah berikut ini.

          
pip install virtualenv

Selanjutnya kita install dependency flask dengan menjalankan perintah berikut ini.

          
pip install Flask

Test Flask

Sekarang kita coba buat aplikasi sederhana untuk mengecek apakah flask sudah berhasil kita install. Silahkan buat folder project nya seperti berikut ini.

          
sudo apt-get update
sudo apt-get install python3.6

Sudah selesai, sekarang cek apakah installasi sudah berhasil dengan menjalankan perintah berikut ini.

          
// Pada windows
md flask-hello-world

// Linux or osx
mkdir flask-hello-world

// Masuk ke dalam direktori
cd flask-hello-world

Sekarang silahkan copykan code dibawah ini untuk membuat aplikasi hello world sederhana simpan dengan nama file app.py.

          
from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
return 'Hello, Flask!'

if __name__ == '__main__':
app.run(debug=True)

Sekarang kita coba jalankan aplikasi sederhana kita dengan menjalankan perintah berikut ini.

          
FLASK_APP=app.py flask run

Selanjutnya silahkan buka http://localhost:5000/
