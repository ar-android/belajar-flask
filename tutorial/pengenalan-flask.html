Apa itu Flask?

Sebelum kita membahas lebih lanjut tentang flask mari kita berkenalan dahulu dengan framework flask. Berikut ini adalah sedikit saya kutip dari website resmi flask :

From Flask website

Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions.

Jadi intinya flask adalah microframework python, yang di maksud dengan microframework adalah kumpulan code dasar yang bisa kita gunaka untuk membuat website dengan bahasa pemrogramman python.

Membuat sebuah website

Bagi kamu yang masih pemula tentu ingin tau alasan nya kenapa kamu harus belajar flask. Nah mari kita bahas sedikit kenapa kita haru belajar flask dan apa untung nya mempelajari flask ini. Jika kamu baru belajar bahasa python maka mungkin kamu akan mendapat rekomendasi dari senior - senior kamu untuk mulai belajar menggunakan framework flask ini.


Ketika membangun sebuah website maka banyak hal yang perlu kamu ketahui, diantaranya memilih bahasa pemrogramman yang akan kamu gunakan. Katakan kamu sudah memutuskan menggunakan bahasa python, sekarang bagaimana melakukanya? Selanjutnya kamu perlu tau hal - hal dasar tentang komponen sebuah website.


Paling tidak kamu harus mengetahui tentang server, karena website kita bisa di akses oleh user itu jika aplikasi web kita ini berjalan di sebuah server. Sudah tau server kamu perlu tau minimal backend dan frontend. Di backend ada beberapa prosess, misalnya pengolahan http request, database, autentikasi, mail dan masih banyak yang lainya. Sedangkan di bidang frontend kamu perlu tau html, css, javascript, responsice design dan yang lain sebagainya.


Bagaimana? Banyak ya hal yang harus kita kerjakan. Ya memang begitulah kenyataanya membangun sebuah website tidaklah begitu mudah. Nah berdasarkan pengalaman saya hal - hal tersebut diatas tidak harus kita pelajari semua, karena itu akan memakan waktu yang cukup lama. Kita bisa pelajari hal - hal tersebut sambil berjalan sambil kita belajar coding, sambil mengalami erorr yang mungkin akan membuat kamu frustasi. Tapi ya begitulah setelah kamu bisa nanti perjuangan belajar akan terasa nikmat.

Kenapa flask?

Diatas sudah sedikit kita bahas cukup panjang mengenai hal - hal dasar yang perlu kita ketahui jika kita ingin membangun sebuah website. Nah dari begitu banyak kerumitan yang harus kita hadapi Flask ini dibuat dengan harapan bisa sedikit mengurangi kerumitan yang ada.


Flask dibagun dengan tujuan kita bisa membuat website dengan mudah dan menyenagkan. Flask akan mengurusi tentang masalah server, orm database, dan juga terdapat berbagai macam extensions untuk memudahkan kita dalam membuat website.

Conton Aplikasi Flask

Berikut ini coba kita lihat contoh aplikasi sederhana dibuat dengan flask.

          
from flask import Flask
app = Flask(__name__)

@app.route("/")
def hello():
  return "Hello World!"

Dan berikut ini cara menjalankan aplikasi yang telah kita buat di atas.

          
$ pip install Flask
$ FLASK_APP=hello.py flask run
 * Running on http://localhost:5000/

Bagaimana simple bukan? Tentu simple ya karena kita hanya membuat aplikasi hello world. Akan tetapi kita akan belajar secara step by step agar mudah dan cepat paham. Nah mari kita lanjutkan pada tutorial berikutnya, kita akan belajar dari nol, jadi jika kamu masih baru dan tidak mengerti banyak hal tentang python kita akan bahas dari nol sampi membangun production apps, mudah - mudahan kamu cepat bisa ya.
