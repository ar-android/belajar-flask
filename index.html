Belajar cara membuat web dengan flask secara struktur.

Pendahuluan

Python saat ini adalah salah satu bahasa pemrogramman yang sangat popular di dalam industri teknologi. Kemudahan sintak dan komunitas yang besar menjadi salah satu alasan python cukup digemari di industri teknologi saat ini.

Flask

Bersama dengan ketenaran python banyak sekali package yang bisa kita gunakan untuk membuat web. Salah satu cara membuat sebuah website dengan python adalah dengan menggunakan package flask. Flask adalah sebuah microframework yang bisa kita gunakan untuk membangun sebuah aplikasi web.

Akan tetapi flask ini tidak memiliki fitur yang cukup lengkap sehingga kita tidak cukup hanya mengandalkan package flask saja untuk membangun keseluruhan web kita. Disini kita akan bahas hal apa saja yang perlu kita ketahui dan lakukan jika ingin menggunakan flask untuk membuat sebuah web.
